# 宿舍管理系统

## 项目简介
基于 Flask + MySQL 实现的宿舍管理系统，包含学生信息管理、宿舍楼宇管理、报修流程管理等模块，支持管理员和学生双角色操作。

## 功能模块
### 学生端
- 注册/登录（身份证号格式校验）
- 查看/修改个人信息
- 提交宿舍维修申请
- 密码修改功能

### 管理员端
- 楼宇信息增删改查（含级联更新）
- 学生信息批量管理
- 维修工单状态管理
- 系统用户密码重置

## 技术栈
- **前端**: HTML5 + CSS3 + JavaScript
- **后端**: Flask (Python 3.8+)
- **数据库**: MySQL 5.7+
- **依赖库**: 
  flask==2.0.3
  pymysql==1.0.2
